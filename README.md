# Smart File Organizer 🧠📁

A Python automation tool that automatically organizes files inside a folder based on file type.

## Features
- Automatically detects file types
- Creates folders if they don't exist
- Moves files into correct categories
- Prevents errors by ignoring folders

## Technologies Used
- Python
- os module
- shutil module

## How It Works
The script scans a selected folder and moves:
- .jpg files → Images folder
- .pdf files → Documents folder
- .exe files → Applications folder
